Box Model
(Блокова модель)

  • Область вмісту (content) — контент елемента (текст, зображення, відео або вкладені елементи). За замовчуванням його висота визначається контентом, ширина - типом елемента (блоковий або рядковий).
  • Внутрішнє поле (padding) — відстань до 'border', використовується для декоративних ефектів. Не можна встановлювати від'ємні значення.
  • Рамка (border) — межа елемента. За замовчуванням - ширина = '0'; колір = кольору контенту (тексту).
  • Зовнішній відступ (margin) — відступ від 'border' до сусідніх елементів. Може приймати від'ємні значення. Фон елемента не поширюється на 'margin'.

width and height properties
Властивості 'width' i 'height'

Ці властивості явно вказують загальну ширину і висоту елемента, або тільки розмір його області вмісту (контенту) — залежить від типу блокової моделі.

Поради:

  • Не ставте елементам фіксовану висоту (може призвести до переповнення) — краще, щоб визначалася контентом.
  • Вказувати одночасно width і height варто тільки декоративним елементам з фіксованим розміром, наприклад іконкам.
  • Можна задавати не тільки в 'px', 'em' або 'rem', але й у '%' — ширина блоку буде розрахована як відсоток від ширини батька (детальніше при побудові сіток).

УВАГА !

Фінальна ширина і висота блоку div.box буде 220px, а не 150px. Існує декілька типів блокової моделі, які відрізняються формулою розрахунку розміру елемента.

box-sizingproperty
Властивість box-sizing

Встановлює тип блокової моделі елемента — формулу розрахунку його розмірів.

box-sizing: content-box | border-box | inherit
  • content-box — властивості width і height задають області вмісту, не включають розміри полів (padding), меж (border) і відступів (margin).

    Цей тип блокової моделі використовується в браузерах за замовчуванням.

    Задавши елементу width 150px, padding 30px і border 5px, отримаємо фінальну ширину 220px.

    /* Фінальна ширина блоку = 150px + 30px + 30px + 5px + 5px */
  • border-box — фінальний розмір елемента, включають значення полів (padding) і меж (border), але не відступів (margin).
  • inherit — наслідує значення блокової моделі предка.
